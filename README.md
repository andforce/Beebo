#Beebo
##Sina Weibo Client
#暂时不支持平板分辨率

##Google Play 上架的版本叫：Beebo+；

##国内市场上线的叫：Beebo；

###两者唯一的区别是：Beebo+可以智能解析微博中的视频链接，并直接播放，因此apk体积会比Beebo大一点。
